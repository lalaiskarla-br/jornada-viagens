# ✈️Jornada Viagens
Interface web responsiva para uma agência de viagens fictícia, desenvolvida com foco em boas práticas de front-end, semântica HTML e experiência do usuário.
___

## 🌍 Sobre o projeto

O Jornada Viagens é um projeto desenvolvido para simular o site de uma agência de turismo, apresentando destinos, pacotes e um formulário de contato.

O objetivo foi praticar:

- Estruturação semântica com HTML
- Estilização com CSS
- Responsividade para diferentes dispositivos
- Organização de arquivos em projetos front-end
